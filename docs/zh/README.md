## 什么是 QuickDeck？

QuickDeck 是一个工具编排平台，允许您：

- **创建工具**：通过可视化界面创建各种自动化工具
- **管理项目**：将工具组织到不同的项目中
- **执行工具**：手动执行或定时执行工具
- **查看历史**：查看工具执行历史和结果
- **管理凭证**：安全地管理各种服务的访问凭证

## 核心功能

### 1. 项目管理
- 创建和管理项目
- 项目成员权限管理
- 项目级别的凭证管理

### 2. 工具创建
- 支持多种运行步骤类型：
  - Bash 命令
  - Shell 脚本
  - Python 脚本
  - CURL 命令
  - MySQL 语句
- 灵活的参数配置
- 支持定时执行
- 支持通知配置

### 3. 执行管理
- 手动执行工具
- 定时自动执行
- 执行历史记录
- 详细的执行日志

### 4. 凭证管理
- MySQL 凭证
- OSS 凭证
- DeepSeek 凭证
- 更多类型支持中...

## 快速开始

1. **部署 QuickDeck** - 参考 [部署说明](./部署说明.md)
2. **创建项目** - 在项目管理页面创建您的第一个项目
3. **创建工具** - 参考 [创建工具](./创建工具.md) 创建您的第一个工具
4. **执行工具** - 在工具列表中执行工具并查看结果
