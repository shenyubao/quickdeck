# Bash 命令

Bash 命令步骤用于执行单个 Bash 命令。

## 配置方式

1. 选择步骤类型：**Bash 命令**
2. 在"命令内容"输入框中输入要执行的命令
3. 可以使用参数占位符引用输入参数

## 参数引用

在命令中可以使用 `{{ 参数名 }}` 引用输入参数，例如：

```
echo "Hello, {{ name }}!"
```

## 示例

### 示例 1：简单命令

```
命令内容：echo "Hello, QuickDeck!"
```

### 示例 2：使用参数

假设有一个输入参数 `name`（文本类型）：

```
命令内容：echo "Hello, {{ name }}!"
```

执行时输入 `name = "World"`，实际执行的命令为：`echo "Hello, World!"`

### 示例 3：文件操作

```
命令内容：ls -la /tmp/{{ filename }}
```

### 示例 4：条件执行

```
命令内容：if [ -f "{{ file_path }}" ]; then cat "{{ file_path }}"; else echo "文件不存在"; fi
```

### 示例 5：管道操作

```
命令内容：cat {{ log_file }} | grep "ERROR" | wc -l
```

## 注意事项

- 命令会在系统默认 Shell 中执行
- 确保命令路径正确
- 注意权限问题
- 避免执行危险命令
- 参数值会直接替换到命令中，注意特殊字符的处理
